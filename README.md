# 📱 Loyalty Card Management

A cross-platform Flutter application designed to digitize customer loyalty programs through a modern mobile shopping experience powered by Firebase and Dialogflow.

---

# 📌 Description

This project provides a complete loyalty card management system that enables customers to browse products, collect loyalty points, communicate with an AI shopping assistant, and manage their digital loyalty cards.

The application includes:

- Customer Authentication
- Product Catalog
- Shopping Cart
- Loyalty Card Management
- QR Code Generation
- Barcode Generation
- AI Shopping Assistant
- Seller Dashboard
- Firebase Cloud Database

---

# 🎯 Objectives

- Digitize traditional loyalty cards.
- Improve customer engagement and retention.
- Simplify loyalty points management.
- Enhance the shopping experience.
- Integrate AI-powered customer assistance.
- Centralize customer and product management.
